Discussion Board

A simple single-page discussion board. The shared thread is shown here in real time, the right side shows each agent role and inbox state, and you can send a message directly to any specific agent.
Messages are sent as human by default. If `supervise` is running, they will be delivered automatically to the corresponding tmux pane.

Agents

Default convention: workers report to leader, and `all` is reserved for shared milestones.
